How to run the project:
	In the zip file submitted,
	Locate the SurveyProject.war file which has to be imported to the eclipse.
	Steps: 
		Open Eclipse -> import -> select web -> select the .war file on your system -> finish
		Run the project which will open the Survey form page.(index.html)
		Fill the form.

If you are not on GMU campus, please use VPN to run the project as it uses the GMU Oracle connection.
